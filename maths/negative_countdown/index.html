Countdown solver

This is a numbers countdown game solver. Below are rules of the game (sligthly modified version of the game show):

  • You are given a set of numbers (both positive and negative numbers are allowed)
  • You are also given a target number
  • The only allowed operations are: `+` (add), `-` (subtract), `*` (multiply) and `/` (divide)
  • All numbers must be used
  • Each number can be used only once
  • Operations can be re-used (for example, two ior more additions in the equation are allowed)
